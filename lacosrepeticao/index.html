Laços de Repetição em C

Introdução

Laços de repetição são estruturas que permitem a execução repetida de um bloco de código enquanto uma condição específica for verdadeira. Em C, os principais tipos de laços de repetição são for, while e do-while. Eles são essenciais para a realização de tarefas repetitivas de forma eficiente.

Estrutura for

A estrutura for é utilizada quando se sabe com antecedência o número de vezes que um bloco de código deve ser repetido. Ela é composta por três partes: inicialização, condição e incremento.

#include <stdio.h>

int main() {
    for (int i = 0; i < 10; i++) {
        printf("Número: %d\n", i);
    }
    
    return 0;
}

Estrutura while

A estrutura while repete um bloco de código enquanto uma condição específica for verdadeira. A condição é verificada antes de cada execução do bloco de código.

#include <stdio.h>

int main() {
    int i = 0;
    
    while (i < 10) {
        printf("Número: %d\n", i);
        i++;
    }
    
    return 0;
}

Estrutura do-while

A estrutura do-while é similar à estrutura while, mas a condição é verificada após a execução do bloco de código, garantindo que o bloco seja executado pelo menos uma vez.

#include <stdio.h>

int main() {
    int i = 0;
    
    do {
        printf("Número: %d\n", i);
        i++;
    } while (i < 10);
    
    return 0;
}

Exemplos de Código

Exemplo com for

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("Quadrado de %d é %d\n", i, i * i);
    }
    
    return 0;
}

Exemplo com while

#include <stdio.h>

int main() {
    int i = 1;
    
    while (i <= 5) {
        printf("Cubo de %d é %d\n", i, i * i * i);
        i++;
    }
    
    return 0;
}

Exemplo com do-while

#include <stdio.h>

int main() {
    int i = 1;
    
    do {
        printf("Número: %d\n", i);
        i++;
    } while (i <= 5);
    
    return 0;
}
