Componentes Básicos do Arduino

Introdução

Os componentes básicos do Arduino são os elementos essenciais utilizados para criar circuitos simples e avançados. Eles incluem LEDs para indicar status, botões para interação do usuário e resistores para limitar corrente e proteger os componentes.

LED

O LED (Light Emitting Diode) é um componente que emite luz quando uma corrente elétrica passa por ele. No Arduino, os LEDs são comumente usados para indicar o estado de um circuito ou para fornecer feedback visual.

Botão

Os botões são usados para fornecer entrada ao Arduino. Eles são dispositivos simples que, quando pressionados, conectam dois terminais elétricos, permitindo que a corrente flua. No Arduino, os botões são frequentemente usados para acionar ações específicas do usuário.

Resistor

Os resistores são componentes usados para limitar o fluxo de corrente em um circuito. Eles são fundamentais para proteger componentes sensíveis e para garantir que os LEDs e outros dispositivos não recebam uma corrente elétrica excessiva, o que pode danificá-los.

Exemplos de Código

Blink com LED

O exemplo Blink é adaptado para acender e apagar um LED conectado ao Arduino em intervalos regulares.

int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH);
  delay(1000);
  digitalWrite(ledPin, LOW);
  delay(1000);
}

Ler estado do Botão

Este exemplo mostra como ler o estado de um botão e imprimir o estado no monitor serial.

int buttonPin = 2;
int buttonState = 0;

void setup() {
  pinMode(buttonPin, INPUT);
  Serial.begin(9600);
}

void loop() {
  buttonState = digitalRead(buttonPin);
  Serial.println(buttonState);
  delay(100);
}
