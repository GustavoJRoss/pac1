Declaração de Variáveis em C

Introdução

Variáveis são elementos fundamentais em qualquer linguagem de programação. Elas são usadas para armazenar dados que podem ser manipulados e utilizados ao longo da execução do programa. Em C, a declaração de variáveis exige que seja especificado o tipo de dado que será armazenado.

Tipos de Variáveis

Na linguagem C, as variáveis podem ser de diferentes tipos, cada um adequado para armazenar um tipo específico de dado. Alguns dos tipos mais comuns são:

  • int: Armazena números inteiros.
  • float: Armazena números de ponto flutuante de precisão simples.
  • double: Armazena números de ponto flutuante de precisão dupla.
  • char: Armazena um único caractere.
  • void: Tipo especial usado principalmente em ponteiros e funções.

Regras de Nomeação

Os nomes das variáveis devem seguir certas regras e convenções para serem válidos:

  • Devem começar com uma letra ou um sublinhado (_).
  • Podem conter letras, dígitos e sublinhados.
  • Não podem ser palavras-chave da linguagem C.
  • Devem ser descritivos para facilitar a compreensão do código.

Escopo e Duração

O escopo de uma variável refere-se à parte do programa onde a variável pode ser acessada. A duração de uma variável refere-se ao tempo durante o qual a variável retém seu valor. Existem três tipos principais de escopo e duração em C:

  • Variáveis Locais: Declaradas dentro de funções ou blocos de código e só podem ser acessadas dentro desses contextos.
  • Variáveis Globais: Declaradas fora de todas as funções e podem ser acessadas por qualquer parte do programa.
  • Variáveis Estáticas: Podem ser locais ou globais, mas retêm seus valores entre chamadas de função.

Exemplos de Código

Declaração e Inicialização

#include <stdio.h>

int main() {
    int idade = 25;
    float altura = 1.75;
    char inicial = 'A';
    
    printf("Idade: %d\n", idade);
    printf("Altura: %.2f\n", altura);
    printf("Inicial: %c\n", inicial);
    
    return 0;
}

Escopo de Variáveis

#include <stdio.h>

int global = 10;

void funcao() {
    int local = 5;
    printf("Variável local: %d\n", local);
    printf("Variável global: %d\n", global);
}

int main() {
    funcao();
    printf("Variável global no main: %d\n", global);
    return 0;
}

Variáveis Estáticas

#include <stdio.h>

void funcao() {
    static int contador = 0;
    contador++;
    printf("Contador: %d\n", contador);
}

int main() {
    funcao();
    funcao();
    funcao();
    return 0;
}
