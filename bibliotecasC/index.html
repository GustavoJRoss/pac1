Bibliotecas da Linguagem C

Introdução

As bibliotecas da linguagem C são conjuntos de funções e macros que facilitam o desenvolvimento de programas ao fornecer funcionalidades prontas para uso. As bibliotecas padrão são distribuídas junto com os compiladores de C e abrangem uma ampla gama de operações, desde manipulação de strings até operações matemáticas complexas.

Bibliotecas Padrão

A linguagem C possui várias bibliotecas padrão que são frequentemente utilizadas no desenvolvimento de software. Aqui estão algumas das mais importantes:

stdio.h

A biblioteca stdio.h fornece funções para entrada e saída padrão, como printf e scanf.

#include <stdio.h>

stdlib.h

A biblioteca stdlib.h oferece funções para alocação de memória, controle de processos, conversões e outras operações gerais. Exemplos de funções incluem malloc, free, atoi e rand.

#include <stdlib.h>

string.h

A biblioteca string.h contém funções para manipulação de strings, como strlen, strcpy, strcmp e strcat.

#include <string.h>

math.h

A biblioteca math.h oferece funções matemáticas, incluindo operações como sqrt, pow, sin, cos, e tan.

#include <math.h>

time.h

A biblioteca time.h fornece funções para manipulação de data e hora, como time, difftime e strftime.

#include <time.h>

Exemplos de Uso

Exemplo com stdio.h

#include <stdio.h>

int main() {
    printf("Digite um número: ");
    int numero;
    scanf("%d", &numero);
    printf("Você digitou: %d\n", numero);
    return 0;
}

Exemplo com stdlib.h

#include <stdlib.h>
#include <stdio.h>

int main() {
    int *array = malloc(5 * sizeof(int));
    if (array == NULL) {
        printf("Falha na alocação de memória\n");
        return 1;
    }
    for (int i = 0; i < 5; i++) {
        array[i] = i * i;
    }
    for (int i = 0; i < 5; i++) {
        printf("%d ", array[i]);
    }
    printf("\n");
    free(array);
    return 0;
}

Exemplo com string.h

#include <string.h>
#include <stdio.h>

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    strcat(str1, str2);
    printf("Concatenated string: %s\n", str1);
    return 0;
}

Exemplo com math.h

#include <math.h>
#include <stdio.h>

int main() {
    double result = sqrt(25.0);
    printf("A raiz quadrada de 25 é: %.2f\n", result);
    return 0;
}

Exemplo com time.h

#include <time.h>
#include <stdio.h>

int main() {
    time_t t = time(NULL);
    struct tm tm = *localtime(&t);
    printf("Data e hora atual: %d-%02d-%02d %02d:%02d:%02d\n",
           tm.tm_year + 1900, tm.tm_mon + 1, tm.tm_mday,
           tm.tm_hour, tm.tm_min, tm.tm_sec);
    return 0;
}
