Introdução ao Arduino

Introdução

O Arduino é uma plataforma de prototipagem eletrônica de código aberto baseada em hardware e software flexíveis e fáceis de usar. É amplamente utilizado por estudantes, artistas, programadores e profissionais em diversos projetos interativos, desde arte interativa até controle de dispositivos.

Hardware

O coração do Arduino é um microcontrolador, geralmente da família AVR da Atmel ou da família ARM. O Arduino originalmente foi construído em torno do ATmega328, mas existem várias placas com diferentes microcontroladores e recursos, adequados para diferentes tipos de projetos.

Software

O ambiente de desenvolvimento integrado (IDE) do Arduino é onde você escreve, compila e carrega o código para o Arduino. É uma interface simples e fácil de usar, adequada para iniciantes e avançados. O código é escrito em uma linguagem de programação baseada em C/C++.

Exemplos de Código

Blink

O exemplo Blink é o "Hello World" do Arduino. Ele faz com que um LED conectado ao Arduino pisque em um intervalo de tempo específico.

int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH);
  delay(1000);
  digitalWrite(ledPin, LOW);
  delay(1000);
}

Button

O exemplo Button demonstra como ler o estado de um botão conectado ao Arduino e responder a ele. Quando o botão é pressionado, um LED é aceso.

int buttonPin = 2;
int ledPin = 13;
int buttonState = 0;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT);
}

void loop() {
  buttonState = digitalRead(buttonPin);

  if (buttonState == HIGH) {
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }
}
