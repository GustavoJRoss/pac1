Comandos de Entrada e Saída em C

Introdução

Os comandos de entrada e saída são fundamentais em qualquer linguagem de programação, pois permitem a interação do programa com o usuário e a manipulação de dados externos. Em C, esses comandos são realizados principalmente através das funções da biblioteca padrão stdio.h.

Comandos de Entrada

Os comandos de entrada permitem que o programa receba dados do usuário ou de outras fontes externas. Os comandos mais comuns são:

  • scanf: Lê dados da entrada padrão (normalmente o teclado).
  • gets: Lê uma linha inteira de texto da entrada padrão (deprecated, use fgets).
  • fgets: Lê uma linha de texto de um fluxo de entrada, recomendado para evitar problemas de buffer overflow.
#include <stdio.h>

int main() {
    int idade;
    printf("Digite sua idade: ");
    scanf("%d", &idade);
    printf("Você digitou: %d\n", idade);
    return 0;
}

Comandos de Saída

Os comandos de saída permitem que o programa envie dados para o usuário ou para outras fontes externas. Os comandos mais comuns são:

  • printf: Escreve dados na saída padrão (normalmente a tela).
  • puts: Escreve uma string seguida por uma nova linha na saída padrão.
  • fprintf: Escreve dados em um fluxo de saída específico.
#include <stdio.h>

int main() {
    int idade = 25;
    printf("Idade: %d\n", idade);
    puts("Esta é uma mensagem com puts.");
    return 0;
}

Exemplos de Código

Exemplo com scanf e printf

#include <stdio.h>

int main() {
    float altura;
    printf("Digite sua altura em metros: ");
    scanf("%f", &altura);
    printf("Sua altura é: %.2f metros\n", altura);
    return 0;
}

Exemplo com fgets e puts

#include <stdio.h>

int main() {
    char nome[50];
    printf("Digite seu nome: ");
    fgets(nome, 50, stdin);
    printf("Olá, ");
    puts(nome);
    return 0;
}

Exemplo com fprintf

#include <stdio.h>

int main() {
    FILE *file = fopen("saida.txt", "w");
    if (file == NULL) {
        printf("Erro ao abrir o arquivo!\n");
        return 1;
    }
    fprintf(file, "Esta é uma mensagem escrita no arquivo.\n");
    fclose(file);
    printf("Mensagem escrita no arquivo com sucesso.\n");
    return 0;
}
